# План реализации проекта Intersson
## Обзор проекта
Intersson — это веб-платформа по подписке, предоставляющая IT-услуги и премиальный контент. Проект будет реализован как современное веб-приложение (Next.js) с фокусом на уникальный, "живой" дизайн, вдохновленный предоставленными макетами из Figma (Liquid gradients, Glassmorphism).

## Технологический стек
- **Framework**: Next.js 15 (App Router)
- **Язык**: TypeScript
- **Стилизация**: Vanilla CSS (CSS Variables + Scoped CSS Modules) для максимальной производительности и гибкости анимаций.
- **UI Компоненты**: Собственная библиотека компонентов на основе Figma (без использования сторонних UI-китов для сохранения уникальности).
- **Анимации**: CSS Keyframes + возможно framer-motion для сложных переходов.

## Основные функции (MVP)
1.  **Лендинг**:
    -   Главный экран с "жидким" анимированным фоном (Liquid Background).
    -   Презентация услуг (карточки с эффектом стекла).
    -   Секция тарифов/подписки.
2.  **Личный кабинет** (Dashboard):
    -   Обзор активной подписки.
    -   Доступ к контенту/услугам.
3.  **Аутентификация** (Заглушка/Планируется):
    -   Вход/Регистрация.

## Дизайн-система (из Figma)
-   **Цвета**:
    -   Основные фоны: Черный (#000000), Темно-серые оттенки.
    -   Акценты: Градиенты (Liquid Shapes), Белый текст.
-   **Типографика**: Использование шрифта `Inter`.
-   **Эффекты**: Blur (размытие фона), Glassmorphism (полупрозрачные карточки).
